# 네이버 뉴스 스크랩 앱 - 윈도우 사용법

## 📋 개요
이 앱은 네이버 뉴스에서 키워드별로 기사를 검색하고 정리해주는 도구입니다.
코스맥스, 화장품 업계, 건강기능식품 관련 뉴스를 모니터링할 수 있습니다.

## 🚀 시작하기

### 1단계: 필수 프로그램 설치
윈도우에서 앱을 실행하기 위해 다음 프로그램들이 필요합니다:

**Python 설치 (필수)**
- 🔗 https://www.python.org/downloads/
- Python 3.8 이상 버전 설치
- 설치 시 "Add Python to PATH" 체크박스 꼭 선택하기

**Node.js 설치 (필수)**
- 🔗 https://nodejs.org
- LTS 버전 다운로드 및 설치 (안정적인 버전)

### 2단계: 앱 설치
1. `install_windows.bat` 파일을 **더블클릭**하여 실행
2. 자동으로 필요한 패키지들이 설치됩니다
3. "설치가 완료되었습니다!" 메시지가 나오면 완료

### 3단계: 앱 실행
1. `start_app.bat` 파일을 **더블클릭**하여 실행
2. 자동으로 웹 브라우저가 열립니다
3. http://localhost:3000 주소로 접속됩니다

## 📖 사용법

### 기본 사용법
1. 웹 페이지에서 검색하고 싶은 키워드 입력
2. 검색 버튼 클릭
3. 결과를 확인하고 정리된 기사 목록 확인

### 기본 모니터링 키워드
**화장품 관련:**
- 코스맥스, 코스맥스엔비티, 콜마, HK이노엔
- 아모레퍼시픽, LG생활건강, 올리브영
- 화장품, 뷰티

**건강기능식품 관련:**
- 건강기능식품, 펫푸드, 마이크로바이옴
- 식품의약품안전처

## 🛑 앱 종료하기
1. `stop_app.bat` 파일을 **더블클릭**하여 실행
2. 또는 실행 중인 검은 창들을 모두 닫기

## 🔧 문제 해결

### 앱이 실행되지 않을 때
1. `stop_app.bat` 실행하여 완전 종료
2. 컴퓨터 재시작
3. `start_app.bat` 다시 실행

### Python/Node.js 설치 확인
1. 윈도우 키 + R 누르기
2. `cmd` 입력하고 엔터
3. `python --version` 입력하고 엔터
4. `node --version` 입력하고 엔터
5. 버전이 표시되면 정상 설치됨

### 브라우저가 자동으로 열리지 않을 때
- 수동으로 브라우저에서 http://localhost:3000 접속

### 포트 충돌 문제
- 다른 프로그램이 3000번 포트를 사용 중일 수 있습니다
- 컴퓨터를 재시작하고 다시 시도해보세요

## 📁 파일 설명
- `install_windows.bat`: 최초 설치용 스크립트
- `start_app.bat`: 앱 실행 스크립트  
- `stop_app.bat`: 앱 종료 스크립트
- `WINDOWS_사용법.txt`: 이 사용법 파일
- `backend/`: Python 백엔드 서버
- `frontend/`: React 프론트엔드 앱
